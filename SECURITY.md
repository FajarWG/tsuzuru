# Security Policy

## Supported Versions

We actively support security updates for the latest version of Tsuzuru.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please **do not open a public issue**. Instead, please report it securely:

1. **GitHub Security Advisory**: Go to the **Security** tab of this repository on GitHub and select **Report a vulnerability**. This is the preferred method for secure disclosure.
2. **Direct Contact**: Alternatively, you can reach out directly to the maintainer via GitHub contact options or open a private draft security advisory.

We take security seriously. Once a vulnerability is reported, we will acknowledge receipt within 48 hours and coordinate a fix and release timeline before public disclosure.
